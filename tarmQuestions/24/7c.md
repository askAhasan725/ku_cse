Here is a short and clear explanation of **`pow()`** and **`abs()`** in C—easy to understand.

---

# ✅ **pow()**

**Header:**

```c
#include <math.h>
```

### ✔ What it does

`pow(a, b)` calculates:

```
a^b   (a raised to the power b)
```

### ✔ Examples

```c
pow(2, 3);   // 8
pow(5, 2);   // 25
pow(9, 0.5); // 3 (square root)
pow(16, 0.25); // 2 (4th root)
```

### ✔ Important notes

* `pow()` always returns a **double**.
* Must compile with math library:

  ```
  gcc file.c -lm
  ```
* Use `sqrt()` if you only need square root—it is faster.

---

# ✅ **abs()**

**Header:**

```c
#include <stdlib.h>
```

### ✔ What it does

`abs(n)` returns the **absolute value** of an integer.

```
abs(-5) = 5
abs(5)  = 5
abs(0)  = 0
```

### ✔ Examples

```c
int x = abs(-12);   // 12
int y = abs(20);    // 20
```

---

# ⚠️ **Common confusion**

There are TWO versions:

### 1. For integers

```c
abs(int x)
#include <stdlib.h>
```

### 2. For floating point

```c
fabs(double x)
#include <math.h>
```

Example:

```c
fabs(-3.14);  // 3.14
```

---

# ✔ Summary

| Function    | Meaning                   | Example            |
| ----------- | ------------------------- | ------------------ |
| `pow(a, b)` | a raised to power b       | `pow(3, 4) = 81`   |
| `abs(n)`    | absolute value of integer | `abs(-7) = 7`      |
| `fabs(n)`   | absolute value of double  | `fabs(-2.5) = 2.5` |
